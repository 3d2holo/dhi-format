# dhi-format

[English Version](./README.en.md)

`dhi-format` 是一种面向数字全息与多视角显示场景的图片格式，用于存储一个或多个观察角度下的图像数据，并支持 `Alpha` 通道。它适合用于需要透明叠加、空间视差和多角度内容表达的应用，例如 `VR/AR`、数字全息显示、裸眼 `3D`、以及 `3D lenticular` 光栅立体图像等。

## 简介

传统图片格式通常只描述单张二维图像，而在全息显示、多视角显示和空间计算场景中，往往需要同时保存多个视角下的画面内容。`dhi-format` 的目标就是提供一种更适合这类场景的图像封装方式：

- 可以只保存单个角度的图像，兼容简单展示需求
- 也可以保存多个角度的图像，用于表达视差或多视点信息
- 支持 `Alpha` 通道，适合透明背景、图层合成和空间内容叠加
- 适用于从内容生产到终端显示的多种链路

## 核心能力

### 1. 单视角与多视角统一存储

`dhi-format` 既可以表示一张普通图像，也可以表示同一对象或场景在多个观察角度下的图像集合。这样可以让同一种格式同时覆盖以下两类需求：

- 单角度预览图
- 多角度全息图或视点序列

对于上层应用来说，这意味着可以使用统一的数据结构处理从简单静态图到复杂多视角内容的不同场景。

### 2. 支持 Alpha 通道

`dhi-format` 支持透明度信息，可用于：

- 前景对象与背景分离
- 在 `AR` 场景中叠加虚拟内容
- 多层图像合成
- 保留全息元素边缘细节，减少硬边和遮挡问题

相比只支持不透明像素的数据格式，`Alpha` 支持使其更适合空间显示和交互式渲染流程。

### 3. 面向空间显示与视差表达

多视角数据可以用于表达观察者位置变化时的图像差异，从而支持：

- 视差效果
- 立体感增强
- 多观察方向下的显示适配
- 裸眼 `3D` 或光栅显示设备的数据输入

这使得 `dhi-format` 不只是“图片文件”，更适合作为空间内容的一种基础承载格式。

## 适用场景

`dhi-format` 适合以下领域：

- `VR/AR` 内容资源存储
- 数字全息显示与全息素材管理
- 多视点渲染结果封装
- `3D lenticular` 光栅立体图像生成与分发
- 裸眼 `3D` 展示内容生产
- 需要透明背景的空间图像素材库

## 数据组织思路

一个 `dhi-format` 文件可以包含：

- 基础元数据
- 一个或多个视角图像帧
- 每个视角对应的角度信息或视点索引
- 图像尺寸、颜色信息与 `Alpha` 信息

一个典型逻辑结构可以理解为：

1. 文件头：标识格式版本、编码方式和总体信息
2. 元数据区：记录尺寸、通道数、视角数量、角度参数等
3. 图像数据区：保存一个或多个视角的像素内容
4. 扩展区：预留给后续的压缩、校准信息、设备参数或渲染参数

这种设计便于后续扩展，也方便根据不同显示设备添加定制化参数。

## 设计目标

`dhi-format` 的设计目标包括：

- 用统一格式描述单视角与多视角图像内容
- 保留透明信息以适配空间合成流程
- 面向全息、`AR`、裸眼 `3D` 等实际显示场景
- 保持结构清晰，便于解析、生成和扩展
- 为未来接入压缩、设备标定和渲染参数预留空间

## 与传统图片格式的区别

与常见的 `PNG`、`JPEG`、`WebP` 等图片格式相比，`dhi-format` 更强调多视角和空间显示能力：

- 传统格式主要面向单张二维图像
- `dhi-format` 可原生组织多个观察角度的数据
- `dhi-format` 更适合服务于全息、立体显示和空间计算链路
- 在支持 `Alpha` 的前提下，可更自然地接入合成与渲染系统

## 示例理解

可以将 `dhi-format` 理解为：

- 当只包含一个视角时，它是一张支持透明通道的空间图像
- 当包含多个视角时，它是一组按照观察角度组织起来的多视图图像集合

例如：

- 一个产品素材可以只保存正视角图像
- 一个全息展示资源可以同时保存左视角、正视角、右视角等多个视图
- 一个 `AR` 对象贴图既可以包含透明背景，也可以包含针对不同观察方向优化的图像内容

## 未来可扩展方向

`dhi-format` 后续可以进一步扩展支持：

- 图像压缩与无损/有损编码策略
- 更丰富的视角描述方式
- 深度信息或法线信息附带存储
- 设备标定参数
- 显示终端适配参数
- 动态多视角序列

## 总结

`dhi-format` 是一种为多视角全息图像设计的图片格式。它既支持单角度图像，也支持多个角度的图像集合，并支持 `Alpha` 通道，适用于 `VR/AR`、数字全息、裸眼 `3D`、`3D lenticular` 等需要空间显示和多视点表达的领域。
